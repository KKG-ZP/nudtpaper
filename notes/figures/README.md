# Figures Directory

本目录用于存放论文笔记引用的图片、表格截图或手工绘制示意图。

建议约定：

- 每篇笔记的图片文件名以论文 key 或笔记文件名前缀开头。
- 优先使用 `pdf`、`png`、`jpg` 等 TeX Live 常见格式。
- 若图片来自论文原文，建议在笔记正文或文件名中保留来源信息，避免后续混淆。

命名示例：

- `brainfm2026_framework.pdf`
- `brainfm2026_results_table.png`
- `sleepfm_ablation.jpg`

若单篇图片很多，可以后续扩展为更细的子目录，但不建议一开始过度拆分。
